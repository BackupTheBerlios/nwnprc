:: Installation :: Guide ::




At this point, you should have the basic PRC installed into the original campaigns.

This means:

75+ new prestige classes

New feats and spells

Epic Spell System, with lots of new epic spells

Access to dozens of Persistent Worlds that are using the PRC

 

However, there are some additional features which need extra installation:

New races

New base classes at level 1

 

These are enabled using the java Character Creator, and that is covered over the next few pages.



Since the java Character Creator is powered by java, the first step is to ensure that java is setup and installed on your PC.

The first step is to go to http://www.java.com It should look something like this:



From there , you need to click on the yellow button in the top right, here shown as "Looking for the Java Virtual Machine?" but the exact details of that page change.

That will then start a process to detect and download java to your PC. It may look like this:



Note the blue shield just below the address bar at the top of the screen. This is windows XP SP2 trying to protect you, simply click on the yellow bar and allow the control to proceed by selecting "Install" from the box that appears.



Once you have started installing java, it should look like this:



Make sure that you read the license agreement and accept it before clicking "Next >"

The next screen looks like this:



A typical installation is fine for most PCs, so simply click "Next >" to continue.

One the process has finished, it will look like this.



Press the "Finish" button to end the installation and then you can move on to the next step.

Now you can go and get the Character Creator itself.

As with the PRC, the place to start is NWVault, but this time enter Character Creator into the search box.



This will get a result set like this:



Though the PRC Character Creator appears several times, they all point to the same entry. Pick one and click on it.

Then you will have a window like this:



As with the main PRC download, select Download File and save it to your desktop.

The java CC is distributed using a compression system called RAR. Usually the program to open it is WinRar, which is covered in the next step.

Now you have the java CC downloaded to your computer but in a compressed format.

If you don't have it already, I suggest you get WinRAR.

To start with go to http://www.rarlabs.com



Then go to the downloads page via the link on the left of the screen.



The normal version you need is the top one on the list, WinRAR 3.42

Again, you will get a warning message from windows.



Select "Save" and then place it on your desktop.

After it has finished downloading, you will end up with your desktop looking something like this:



The next step is to start the installation of WinRAR by double-clicking on the wrar342.exe program.

Windows may warn you again about the program with a message like this:



As with the main PRC package, select "Run" if you are confident that the file contains no viruses.

The first install screen looks like this:



Make sure that the destination folder is correct and that you agree to the agreement.

WinRAR is shareware, if you continue to use it after 40 days, you must register it for a small fee.

After the "Install" button is clicked, it will install WinRAR onto your PC.

Then it will prompt you as to which options you wish to use:



Once the settings are tailored to your satisfaction, press the "OK" button to complete the installation.

Now that you have WinRAR installed, you can extract the java CC from its distribution package.

To do so, simply right click on the file on your desktop and select "WinRAR" then "Extract Here".



This will extract a pair of files, cc.jar and RUN_CC.bat.

The "cc.jar" file is the java Character Creator program itself.

The "RUN_CC.bat" is a batch file you should use to run the Character Creator.

By now you should have the java CC file "cc.jar" and the batch file "RUN_CC.bat" extracted from the rar and sitting on your desktop.

To start the java CC, simply doubleclick on the "RUN_CC.bat" file.

Congratulations! At this stage you should have the java CC running correctly.

Now that the java Character Creator is installed and working correctly, you should have a screen like this:



The first thing you need to do is to tell it to use the PRC files.

 To do so, press the "Settings" button and setup the next screen so that it looks like this:



The directories will vary depending where you installed NWN.

Once the settings are correct, press the "OK" button to return to the main screen.

Back at the main screen, press the "Create Character" button to start to make a character.



The java Character Creator is designed to replicate biowares in-game character creator.

As such, you go through the same stages as normal in creating your character.

One change is at the potraits stage.



Here there are two checkboxes on the left of the screen.

These restrict which portraits are displayed, based on gender or race.

If you are using a new PRC race, you will have to deselect "Lock to Race" in order to see any portraits.

Once you have reached the end of the process, press "Finalize" to create the character.

And once you have made your character, you can select it in-game by pressing "Use a pre-made character" and selecting it from the list that appears.
