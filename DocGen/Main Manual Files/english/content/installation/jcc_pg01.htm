At this point, you should have the basic PRC installed into the original campaigns.

This means:

75+ new prestige classes

New feats and spells

Epic Spell System, with lots of new epic spells

Access to dozens of Persistent Worlds that are using the PRC

 

However, there are some additional features which need extra installation:

New races

New base classes at level 1

 

These are enabled using the java Character Creator, and that is covered over the next few pages.
