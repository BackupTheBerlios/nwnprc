By now you should have the java CC file "CC.jar" extracted from the rar and sitting on your desktop.

Now you need to make sure that it is run as a java file, rather than extracted with WinRAR or anything else.

To do so, right click on "CC.jar" and select "Open With" and then "Choose Program...."



This will open a window where you can select what program to run it with.



To be on the safe side, select "Browse..."



Then you need to naviage to where java installed itself. Usually this is in Program Files / Java / [some version number]

Inside the bin directory there should be a file named javaw.exe



Select that file and then press the "Open" button.

Then select the "OK" button to open the file.

Congratulations! At this stage you should have the java CC running correctly.
