The next step is to use the automatic patching process to make sure you have the most up-to-date version of NwN.

Simple start NWN menu from the start menu, or by inserting the CD. You will see a screen like this



Note carefully the number in the top left of that window. If it says 1.59 or anything less than 1.65, then you have an older version and should click the update button to launch the automatic updater.

Even if you have a later version, then its probably worth checking the automatic updater every so often to make sure you stay up-to-date.

If you are workign from a fresh install of NWN + SoU + HotU, youll probably see a screen like this:



Press the Update button to start the process. Make sure you are connected to the internet at this point.

After a while, you may see a message like this:



If you do, simply click OK and wait.

Once the updater has finished doing its thing, you should see something like this:



You might also be presented with options to read Bioware newsletters or other important information.

Read them or not as you desire.
