Now that you have the latest version of NWN, you need to get the PRC pack.

Start by going to the Neverwinter Vault at http://nwvault.ign.com You should be presented with a screen like this:



You need to enter "Player Resource Consortium" into the search box and the press the submit button.

After a while, the results of the search will come back, they should look like this if you scroll down a bit:



The entry you want is the "Player Resource Consortium's PRC Pack v2.2c"

The exact version number may change as newer editions are released

Once you have clicked on that link, you should be able to scroll down and see a screen like this:



Here, the important thing you need is the "Download File" link.

If you like the PRC, feel free to come back here and vote!

Once you start the download process you might see a window like this appear.

If you don't see exactly the same window, do not panic. It varies depending on what version of windows you have installed.



You need the "Save" or "Save file to disk" option.

For convenience, save the file to your desktop.



Again, exactly what is in the window will vary depending what is on your PC.
