Once you've installed the PRC, you have to actually add it to modules.

Directly after the installer the Module Updater should be launched automatically.

If you want to run it again at a later date, you should have a desktop icon:



(The one labelled "PRC Module Updater in the bottom right of that picture)

And a start menu icon as well:



 

However the Module Updater is launched, you should have a window like this appear:



Simply tick the boxes next to the module you wish to install the PRC in and then press the "Install" button.

For starters, I would suggest selecting the "Neverwinter Original Campaign", "Shadows of Undrentide Campaign", and "Hordes of the Underdark Campaign".

If you have other modules, ones you have created or downloaded from NWVault or elsewhere, you can also install the PRC in them. However, we cannot guarantee that it will be compatible and work without any unforeseen problems.

As the Module Updater works away at installing the PRC into your modules, you may see an error message like this:



This warns you about conflicts between the PRC and the module you are trying to install the PRC into.

Most of the time it is safe to press the "Continue" button, however, if you then experience problems with the module these warnings are a good place to start tracking down the problem.
