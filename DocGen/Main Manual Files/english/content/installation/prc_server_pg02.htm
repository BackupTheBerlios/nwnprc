The first step is to download the PRC Server Pack from here. If your using windows, you should get the .exe installer. At this time there is not a linux or mac compatible version.
To install the PRC Server Pack, simply run the exe. It should automatically detect your NWN directory. After it has installed itself, it will offer to precache the 2das for you straght away. If you need to precache it again in the future, run the "precache sqlite.bat" file in your NWN directory. One word of warning, that batch file will delete the old database, so if you have systems that modifiy it in some way you may want to look at alternatives or move data around.
Once the precacher has finished, you should have a ready populated database.
