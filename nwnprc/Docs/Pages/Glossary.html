Glossary of terms

back to Introduction.

Here is a glossary of common terms, definitions, and abreviations.

PRC
Prestige Class Consortium.

ESS
Boneshanks Epic Spellcasting System.

PrC
Prestige Class.
see prestige class.

NWN
Nevewinter Nights.
The computer game upon which this is all reliant.

SoU
Shadows of Undrentide.
The first expansion pack to Neverwinter Nights.

HotU
Hordes of the Underdark.
The second expansion pack to Neverwinter Nights.

.erf
A file type that can be exprted or imported from the toolset. Similar to a zip file in that an erf contains lots of other files within it. .hak and .erf are interchangeable so you can rename one to get the other and visa versa.

.hak
A file type used to contain custom content. Has to be added to a module to be used. Similar to a zip file in that an hak contains lots of other files within it. .hak and .erf are interchangeable so you can rename one to get the other and visa versa.

.mod
A module file type. This is the main file type that the game and toolset load to play/edit an adventure. Similar to a zip file in that an mod contains lots of other files within it.

.nwm
This is the extension given to the .mod files containing the origianl campaigns.
see .mod.

.hif
A file type used by bleedingedge's module updater to determine what files to add and merge. In plain text format so it can be opened with notepad or similar.

.2da
A 2-dimensional array. A file type used by bioware to store data. Normally contained inside .hak files.

.nss
A file type used for a script source code. Normally contained inside .mod files, these are also inside the PRC .haks. In plain text format so it can be opened with notepad or similar.

.ncs
A file type used for compiled script source code. Normally contained inside .mod files, these are also inside the PRC .haks. These are generated by a compiler, ususally the toolset, from .nss files to be read by the game.

prestige class
A class that cannot be taken as the first class, usually due to specific prerequisites.
